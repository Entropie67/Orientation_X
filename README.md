# Horizon Orientation

## Démarrage rapide

```bash
npm install
npm run dev
```

⚠️ Important: ne pas ouvrir `index.html` directement dans le navigateur (fichier local `file://`).
Le style Tailwind est compilé par Vite/PostCSS : il faut lancer le serveur dev ou faire un build.

## Build production

```bash
npm run build
npm run preview
```
